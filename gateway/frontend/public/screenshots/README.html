PWA Screenshots Needed

This directory holds screenshots shown in the browser's "Install App" prompt and in app store listings for the DingDawg Agent 1 PWA.

These files are not yet generated. Add real screenshots to enable rich install prompts on Chrome/Edge.

Required files

  • screenshot-wide.png — 1280 × 720 px, desktop/wide viewport
  • screenshot-narrow.png — 750 × 1334 px, mobile/narrow viewport

How to generate

  1. Open the app in a browser at the size listed above.
  2. Take a screenshot (or use Playwright's page.screenshot()).
  3. Save the file here with the exact filename above.
  4. Update public/manifest.json to point to these files under the screenshots key.

Playwright snippet

// e2e/generate-screenshots.ts
await page.setViewportSize({ width: 1280, height: 720 });
await page.goto('/');
await page.screenshot({ path: 'public/screenshots/screenshot-wide.png' });

await page.setViewportSize({ width: 750, height: 1334 });
await page.screenshot({ path: 'public/screenshots/screenshot-narrow.png' });
